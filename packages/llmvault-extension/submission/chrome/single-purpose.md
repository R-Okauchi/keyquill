# Single purpose

LLMVault stores a user's LLM API keys in isolated browser storage and lets user-approved web apps use those keys for chat completions and tool-calling, without the hosting web app, its server, or any third party ever seeing the key material.

That is the extension's only purpose. It does not modify web-page content, inject advertising, track browsing, or provide any functionality unrelated to LLM API key custody and request brokering.
