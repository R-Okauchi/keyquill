Keyquill Demo

Your API key stays in the extension. This page never sees it. No server, no logging.

Checking for extension…

Step 1 — Install the extension. Until the Chrome Web Store listing is live, install from source: github.com/R-Okauchi/keyquill.

Step 2 — Register an API key. Click the Keyquill icon in your browser toolbar, add an OpenAI-compatible key. The key is stored in chrome.storage.session and is cleared when you close the browser.

Step 3 — Connect this page. Click below. A consent popup will appear; approve it to let this page use your key.
